# Xtension

## Local Setup
1. `npm install`
1. `npm run start`
1. Navigate to [`chrome://extensions/`](`chrome://extensions/`) in a new tab and toggle "Developer Mode" to on
1. Click load unpacked and select the xtension/dist directory
